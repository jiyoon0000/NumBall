#### 2. 정답을 맞추기 위해 숫자 입력할 수 있게 하기
-서로 다른 세자리 수 입력
-숫자만 입력 가능, 문자 입력할 수 없음


#### 3. 결과 값 출력 및 게임 로직 적용하기
-정답값과 입력값을 비교해 볼,스트라이크, 아웃으로 표시
-스트라이크 : 같은자리에 같은숫자
-볼 : 같은숫자, 다른자리
-아웃 : 정답값에 없는 숫자


#### 4. 게임 이어서 하기
-정답을 맞출때까지 계속 시도할 수 있어야하고, 정답을 맞추면 축하메시지 출력
